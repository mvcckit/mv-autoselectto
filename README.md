# mv-autoselectto

The autoselectto directive selects text inside a textbox up to a specified character.

## Building mv-autoselectto

Once you have your environment setup, run:

    grunt

## A Quick Example

	<html>
		<body>
			<p>
				<input type="text" mv-autoselectto="-">
			</p>
		</body>
	</html>
